# git-visual
Visualizing a Git repository

## Usage:

From the root of this repository, run:

```bash
index.sh <relative path to repo>
```
or
```bash
./index.sh <relative path to repo>
```

For example, if you cloned this repository to `/home/user/git-visual`, run:

```
./index.sh ../<repo i want to visualize>
```

## Prerequisites:

Node.js, npm, and git.
